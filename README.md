🤖 JokeBot – Telegram Joke Bot

JokeBot is a simple and fun Telegram bot built using Node.js that delivers random jokes on demand. Whether you're bored or just need a quick laugh, this bot has you covered 😂

🚀 Features
🎭 Sends random jokes instantly
💬 Responds to user commands
⚡ Fast and lightweight Node.js backend
🔌 Easy to extend with more features
🛠️ Tech Stack
Node.js
Telegram Bot API
(Optional) Express.js for server setup
📦 Installation
Clone the repository:
git clone https://github.com/sameer-droid/joke-bot.git
cd joke-bot
Install dependencies:
npm install
Create a .env file and add your Telegram bot token:
BOT_TOKEN=your_telegram_bot_token_here
▶️ Running the Bot

Start the server:

node index.js

Or if using nodemon:

npx nodemon index.js
🤖 How to Use
Open Telegram
Search for your bot (created via BotFather)
Start the bot and use commands like:
/start - Start the bot
/joke - Get a random joke
